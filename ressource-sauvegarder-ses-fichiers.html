    Comment sauvegarder efficacement ?

    Étape 1 - Faites des sauvegardes régulières et archiver différentes versions de votre fichier.
    • De façon régulière (à chaque petite modification) utiliser l'outil /Fichier/Enregistrer/
    • Lors de modifications plus importantes utiliser l'outil /Fichier/Enregistrer sous/  Changer le nom de votre fichier en ajoutant par exemple "Version2"
    • Une sauvegarde peut mal se réaliser et le fichier sauvegardé sera illisible (corrompu). Le fait de créer plusieurs versions distinctes vous permetra de ne pas tout perdre si votre sauvegarde "corrompt" votre fichier

    Étape 2 - Variez les supports de sauvegarde
    • La sauvegarde de votre travail doit se faire sur le disque dur de l'ordinateur, dans le répertoire de votre choix.
    • Pensez à copier votre dossier sur un autre support (ex: clef USB, Cloud, Email ...)

    Étape 3 - Contrôlez vos copies
    • Quel que soit le mode de sauvegarde choisi, un contrôle régulier s’impose. Vérifiez de temps en temps que les fichiers peuvent s’ouvrir. Un disque dur externe doit être redémarré régulièrement, sous peine de se gripper définitivement.
    • Si, sur votre disque dur externe, certains documents présentent des défauts, il est temps de les recopier tous sur un nouveau support. Si vous possédez le fichier d’origine, sauvegardez-le une nouvelle fois. Dans le cas contraire, essayez de le transférer sur l’ordinateur. Si vous y arrivez, c’est qu’il ne contenait pas d’erreur non corrigible. Il ne reste plus qu’à le sauvegarder à nouveau. 

    Étape 4 - Pour les diaporamas ou autre travaux à rendre
    • Une fois votre travail terminé, exporter ou sauvegarder votre fichier dans un format qui sera compatible avec les logiciels du collège. En cas de doute utilisez le format PDF en plus de votre fichier d'origine.
